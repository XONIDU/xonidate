# Somos XONINDU

# Requisitos mínimos
Asegúrate de tener instalado Python 3 en tu sistema. Luego, instala las dependencias necesarias según tu sistema operativo:

---

## Instalación en **Arch Linux**
```bash
sudo pacman -S python-pip
pip install flask fpdf --break-system-packages
sudo pacman -S tk
```

## Instalación en **Ubuntu y derivados**
```bash
sudo apt update
sudo apt install python3 python3-pip -y
sudo apt install python3-tk python3-dev scrot python3-xlib -y
pip3 install flask fpdf --break-system-packages
```

## Instalación en **Windows**
```bash
pip install flask fpdf
```

## Instalación en **macOS**
(Asegúrate de tener Homebrew instalado: https://brew.sh/)
```bash
brew install python3
pip3 install flask fpdf
```

---

# Ejecución de la aplicación:

Para ejecutar el programa y acceder a la aplicación web, utiliza el siguiente comando:
```bash
python start.py
```
